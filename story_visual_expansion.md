# 一、视觉风格指南

## 1.1 整体风格定位

**"赛博东方"（Cyber-Oriental）**—— 赛博朋克的冷冽科技感与中国古典美学融合。

- 光影：高对比度，霓虹冷光与暖黄灯笼光交织
- 材质：金属/玻璃/全息投影 vs 木质/丝绸/纸灯笼
- 氛围：雨夜、雾气、倒影、多层景深
- 色调：深蓝/紫/青为主色，琥珀/朱红为点缀

## 1.2 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色-夜空 | `#0A0E27` | 背景底色，深邃夜空 |
| 主色-霓虹青 | `#00F5D4` | 数据流、全息投影、Luna的视觉标识 |
| 主色-量子紫 | `#7B2FBE` | 时间裂缝、意识空间、超自然元素 |
| 点缀-琥珀 | `#FFB347` | 暖光、旧城灯火、记忆回溯 |
| 点缀-朱红 | `#E63946` | 危险、警告、情感高潮 |
| 中性-银灰 | `#B8C4D0` | UI界面、WDC官方视觉 |
| 中性-深棕 | `#3D2B1F` | 旧城建筑、木质结构 |

## 1.3 新长安城市设计原则

- **天网区**：垂直都市，玻璃幕墙，全息广告牌，无人机航道，冷色调，秩序感
- **旧城**：水平蔓延，窄巷深院，霓虹招牌，蒸汽管道，暖色调，混乱生机
- **记忆银行**：无限延伸的白色走廊，悬浮的数据光点，庄严冰冷
- **数据废墟**：破碎的建筑漂浮在虚空，褪色的数据碎片如落叶
- **量子裂缝**：空间裂口边缘呈锯齿状，内部可见另一个时间的影像


## 4.2 10个核心视频片段

### 片段1：开场 — 新长安2077（30秒）
- **内容**：航拍新长安天际线，从旧城飞向天网区，展示城市全貌
- **视觉**：黄昏时分，先看到旧城的暖色灯火和窄巷，镜头拉升穿过雾气，天网区的全息广告牌和无人机航道显现，最终定格在WDC总部大楼
- **配乐**：二胡独奏渐入电子合成器
- **旁白**："2077年，新长安。每个人的意识都被编码为一串独一无二的神经码……"

### 片段2：记忆银行内部（20秒）
- **内容**：L3档案区的无限走廊
- **视觉**：第一人称视角，缓步走入白色走廊，两侧意识修复舱的光芒在脸上流转
- **配乐**：空灵的电子音，如心跳般的低频脉冲

### 片段3：莫瀚文的暗巷（25秒）
- **内容**：旧城雨夜，莫瀚文出场
- **视觉**：雨滴特写 → 霓虹招牌倒影 → 暗巷深处脚步声 → 莫瀚文从阴影中半转身
- **配乐**：爵士钢琴 + 雨声
- **莫瀚文台词**："想找走私者？我有线索。但你需要帮我一个忙。"

### 片段4：Luna/Su Nian的身份裂变（35秒）
- **内容**：Luna从AI变为苏念的关键时刻
- **视觉**：蓝色全息女性轮廓 → 裂缝出现 → 温暖的女性面容从裂缝中浮现 → 两张脸在画面中叠加 → 最终融合
- **配乐**：弦乐渐强 → 静默1秒 → 钢琴单音
- **Luna台词**："我……我记得了。我曾是苏念。"

### 片段5：量子裂缝（25秒）
- **内容**：S5核心视觉，时间裂缝出现
- **视觉**：正常街景 → 空间突然扭曲如镜面碎裂 → 裂缝中投射出2071年的影像 → 两个时空在同一画面叠加
- **配乐**：玻璃碎裂音效 + 反向播放的弦乐

### 片段6：林正远的37分钟（30秒）
- **内容**：2071年实验室，林正远按下上传键
- **视觉**：简陋实验室，林正远特写，手中照片，手指悬在按钮上方，最终按下 → 蓝光爆发
- **配乐**：只有呼吸声和心跳 → 按键瞬间音乐爆发
- **旁白**："如果他够勇敢，就不需要这种技术来留住你。"

### 片段7：三个爻（25秒）
- **内容**：意识三分裂变
- **视觉**：一个AI核心光球 → 裂成三份 → Alpha（金色，微笑） → Beta（银色，冷漠） → Gamma（紫色，闭眼）→ 三者对峙
- **配乐**：单一旋律 → 分为三个声部 → 三声部各自演奏不同曲调

### 片段8：守墓人的牺牲（35秒）
- **内容**：守墓人化为锚点
- **视觉**：守墓人摘下兜帽 → 张开双臂 → 身体从边缘开始化为金色粒子 → 碎片被锚定 → 最终只剩微笑的轮廓
- **配乐**：二胡独奏 → 加入大提琴 → 最强的全编制 → 突然只剩二胡最后一个音
- **守墓人台词**（轻声）："够了。这样就够了。"

### 片段9：完美世界的恐怖（25秒）
- **内容**：新长安-B的"天堂"
- **视觉**：完美的城市 → 人们同步行走 → 花朵同时绽放 → 镜头推近一个居民的脸 → 空洞无表情的眼睛 → 背景逐渐扭曲
- **配乐**：甜美但机械的旋律，节拍完全均匀

### 片段10：桥之门（30秒）
- **内容**：桥之门矗立，两个世界连接
- **视觉**：夕阳下的光之拱门 → 两个世界的人们穿行 → 桥的光芒逐渐融入夕阳 → 最终定格在拱门剪影
- **配乐**：全编制管弦乐 → 逐渐简约 → 最后只剩二胡与电子音的对话
